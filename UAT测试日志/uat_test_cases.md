## 📋 运行测试

```bash
# 安装 Playwright 浏览器（首次）
cd frontend && npx playwright install chromium

# 运行全量 UAT（确保后端已启动在 :8000）
npx playwright test tests/e2e/uat.spec.ts --headed

# 只运行某一功能分组
npx playwright test tests/e2e/uat.spec.ts --grep "UAT-F03"

# 生成 HTML 报告
npx playwright test tests/e2e/uat.spec.ts --reporter=html
```
